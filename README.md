# Quotes App

A React-based quotes app with two **Random  Quote**, a single-card view that fetches one random anime quote at a time, showing the poster as a background with a glassmorphic quote card, a hide/show toggle, and an anime source link below; **My Quotes**, a personal collection of over 300 curated quotes from friends, colleagues, and various media displayed in a randomised swipeable carousel format with a static background image.
